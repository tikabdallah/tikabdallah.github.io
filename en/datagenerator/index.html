Data Generator is a free native Salesforce app that lets you create realistic records on any standard or custom object — whether you're testing your developments or impressing during a client demo. No code, no CSV, no friction.

Explore features

Everything you need for testing and demos

Built for Salesforce developers, admins, and QA teams.

⬡

Any Salesforce object

Generate records on all standard and custom objects in your org: Account, Contact, Opportunity, and many more.

◈

Multi-object in one run

Create multiple objects in a single execution and automatically link them via lookup fields — no manual post-processing needed.

≡

Field-by-field control

Configure each field individually: text type, phone format, date range, picklist distribution, blank value rate.

⊞

Reusable templates

Save your configurations as templates and relaunch them in one click, or start from a ready-to-use template to get up and running fast.

↺

One-click delete

Made a mistake? Delete all records generated by a run and relaunch a new configuration immediately.

⟳

Post-sandbox automation

Configure templates that run automatically on every sandbox refresh via the native SandboxPostCopyHandler.

See the app in action

100% native Salesforce Lightning interface.

Template or free setup

Start a new generation from a pre-configured template or build your own setup from scratch. A sample template (Sales Cloud) is included on install.

Object relationships

Add as many objects as needed in a single run. Lookup fields automatically link the created records — for example, associating Contacts with Accounts generated in the same run.

Instant generation & full history

Records are created in seconds. Every run is logged in the History tab with record count, date, and any errors.

Automation on sandbox refresh

Configure your templates to run automatically on every sandbox refresh. Ideal for keeping your test environments ready to use as soon as they're created.
